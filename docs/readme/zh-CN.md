## 📖 项目简介

- 这是一个数据分析老师傅写给不太懂数据分析，但是又有数据分析需求的同学的轻量级工具。帮助他们解决一般数据分析需求，助力他们更好的完成工作或自己的小想法！

---

## ✨ 功能特性

- 📂 功能 1：数据管理
- ⚠️ 功能 2：异常分析
- 👥 功能 3：聚类分析
- 📈 功能 4：预测分析（回归/分类）
- 💹 功能 5：营销组合优化

---

## ⚙️ 安装和使用
```
git clone https://github.com/Darren-Li/Ignite.git
cd Ignite
```

安装python依赖包
```python
pip install -r requirements.txt
```

▶️ 启动
```
streamlit run ignite.py
```
